# Postman Setup Guide — Binance FAPI Testnet Audit

## Quick Start

### 1. Create Postman Collection
- New Collection → Name: "Binance FAPI Testnet Audit"
- Add variables:
  - `base_url`: `https://demo-fapi.binance.com`
  - `api_key`: (copy from `.env` `BINANCE_DEMO_API_KEY`)
  - `api_secret`: (copy from `.env` `BINANCE_DEMO_API_SECRET`)
  - `timestamp`: (auto-generated by pre-request script)
  - `recvWindow`: `5000`

### 2. Pre-request Script (for signed requests)
Add to Collection → Settings → Pre-request Script:

```javascript
// Auto-generate timestamp for signed requests
const timestamp = Date.now();
pm.environment.set("timestamp", timestamp);
pm.environment.set("recvWindow", 5000);

// HMAC-SHA256 signature generator
const generateSignature = (queryString, apiSecret) => {
  const crypto = require('crypto');
  const hmac = crypto.createHmac('sha256', apiSecret);
  hmac.update(queryString);
  return hmac.digest('hex');
};

// Save for request bodies
pm.environment.set("generateSignature", generateSignature.toString());
```

### 3. Request Authentication
For each signed request:
1. **Headers**: Add `X-MBX-APIKEY: {{api_key}}`
2. **URL Params** (for GET) or **Body** (for POST/DELETE):
   ```
   timestamp={{timestamp}}
   recvWindow={{recvWindow}}
   signature=<compute below>
   ```
3. **Compute signature**:
   - Build query string: `symbol=SOLUSDT&timestamp=1234567890&recvWindow=5000`
   - Signature = HMAC-SHA256(query_string, API_SECRET in hex)
   
   **Easy way in Postman**: Use `pm-library/hmac-sha256` collection or compute manually:
   ```javascript
   // In request pre-request script:
   const queryString = "symbol=SOLUSDT&timestamp=" + pm.environment.get("timestamp") + "&recvWindow=5000";
   const crypto = require('crypto');
   const signature = crypto.createHmac('sha256', pm.environment.get("api_secret"))
       .update(queryString)
       .digest('hex');
   pm.environment.set("signature", signature);
   ```

## Troubleshooting

### 400 Bad Request — Invalid Signature
- ✓ Timestamp within ±1000ms of Binance server time
- ✓ Query string format: `key1=value1&key2=value2&key3=value3` (alphabetical, no spaces)
- ✓ Signature = HMAC-SHA256(query_string, API_SECRET) in hexadecimal
- ✓ API key/secret copied correctly (no trailing spaces)

### 403 Forbidden — IP Not Whitelisted
- On testnet: Usually not enforced, but possible if demo account has IP restrictions
- Solution: Check demo account settings at https://testnet.binancefuture.com

### 429 Too Many Requests — Rate Limited
- Testnet limit: 1200 requests/min per endpoint
- Exponential backoff with Retry-After header
- Code already handles this (`binance_client.py:92-100`)

### -4120 — STOP_MARKET Not Supported on Testnet
- Expected behavior
- Use LIMIT fallback (price + stopPrice, reduceOnly=true)
- Code already implements this fallback

### -4046 — Leverage Above Maximum
- Trying to set leverage > bracket max
- Solution: Use leverage ≤ 125 (testnet default)

### -2022 — Reduce Only Reject
- Two reduceOnly orders on same symbol simultaneously
- Solution: Cancel TP before placing SL, or vice versa

## Postman Script: Auto-Signature Generator

Add to collection **Pre-request Script** tab:

```javascript
// Auto-compute HMAC-SHA256 for all signed requests
const symbol = request.url.query.get('symbol') || pm.request.body.params?.symbol || null;
const timestamp = Date.now();

// Build query string from all params
let queryParts = [];
if (pm.request.body.mode === 'formdata') {
  pm.request.body.formdata.forEach(param => {
    if (!param.disabled) {
      queryParts.push(param.key + '=' + param.value);
    }
  });
} else if (pm.request.body.mode === 'urlencoded') {
  pm.request.body.urlencoded.forEach(param => {
    if (!param.disabled) {
      queryParts.push(param.key + '=' + param.value);
    }
  });
}

// Add timestamp if not already there
if (!queryParts.some(p => p.startsWith('timestamp='))) {
  queryParts.push('timestamp=' + timestamp);
}

// Sort and build final query string
queryParts.sort();
const queryString = queryParts.join('&');

// Compute HMAC-SHA256
const crypto = require('crypto');
const signature = crypto
  .createHmac('sha256', pm.environment.get('api_secret'))
  .update(queryString)
  .digest('hex');

// Set environment variable for use in request
pm.environment.set('signature', signature);
pm.environment.set('timestamp', timestamp);

console.log('Query String:', queryString);
console.log('Signature:', signature);
```

## Running the Audit Workflow

**Phase 1: Market Data (5 min)**
```
1. GET /fapi/v1/exchangeInfo → catat filter structures
2. GET /fapi/v1/klines → verify OHLCV format
3. GET /fapi/v1/premiumIndex → check funding rate
```

**Phase 2: Account (5 min)**
```
4. GET /fapi/v2/account → record equity
5. GET /fapi/v2/positionRisk → check positions
6. GET /fapi/v2/openOrders → verify empty (or list any stale orders)
```

**Phase 3: Order Flow (10 min)**
```
7. POST /fapi/v1/order (MARKET BUY) → create position
8. POST /fapi/v1/order (LIMIT TP) → set TP bracket
9. GET /fapi/v2/openOrders → verify TP order exists
10. DELETE /fapi/v1/order → cancel TP order
11. DELETE /fapi/v1/allOpenOrders → close everything
```

**Phase 4: Leverage (3 min)**
```
12. POST /fapi/v1/leverage → set 2× leverage
13. GET /fapi/v1/leverageBracket → view tiers
```

**Total time: ~23 min**
